# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a Vulnerability

Please report vulnerabilities privately through the repository's GitHub Security
tab using the **Report a vulnerability** flow (private vulnerability reporting /
security advisories). Do not open a public issue for a suspected vulnerability.

Reports are handled on a best-effort basis by the maintainer.
